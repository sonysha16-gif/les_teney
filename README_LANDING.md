# Документация лэндинга «КВЕСТ 4 КЛЮЧА»

Данная документация описывает техническую структуру, дизайн-решения и функциональные возможности одностраничного веб-приложения (лэндинга) для проекта «Квест 4 Ключа» Софьи Арих.

---

## 🏗 Технологический стек

- **React 18+**: Библиотека для построения пользовательского интерфейса.
- **TypeScript**: Обеспечение типизации и надежности кода.
- **Tailwind CSS**: Утилитарный CSS-фреймворк для стилизации.
- **Motion (motion/react)**: Библиотека для реализации плавных анимаций и интерактивных переходов.
- **Lucide React**: Набор иконок для визуализации разделов.
- **Vite**: Инструмент сборки и разработки.

---

## 🎨 Дизайн и Визуальная концепция

### Эстетика
Приложение выполнено в стиле **"Dark Luxury / Mystical"**:
- **Цветовая палитра**: Глубокий черный (`#0a0a0a`), золотые акценты (`gold`, `gold-light`) и мягкие белые тексты.
- **Типографика**: Использование шрифтов с засечками (serif) для заголовков (создает ощущение премиальности и классики) и шрифтов без засечек (sans) для основного контента (удобство чтения).
- **Спецэффекты**:
    - **Glow Gold**: Текстовые тени и свечение вокруг золотых элементов.
    - **Backdrop Blur**: Размытие фона для модальных элементов и карточек.
    - **Gradient Overlays**: Мягкие переходы для интеграции изображений в темный фон.

### Разделы (Секции)
1. **Hero Section (Главный экран)**: 
    - Фоновое изображение высокого разрешения с дверями и героиней.
    - Динамичная сетка: Текст разнесен по сторонам для акцента на центральном образе.
    - Подсветка ключевых фраз ("Квест" крупно, "4 Ключа" в одну строку).
2. **Results Section (Блоки результатов)**:
    - Интерактивные блоки с эффектом чередования (reverse).
    - Качественные изображения с эффектом наведения (zoom-in).
    - Список из 4 ключевых сфер: Состояние, Финансы, Отношения, Здоровье.
3. **About Me (Обо мне)**: 
    - Фото специалиста с параллакс-эффектом (sticky-позиционирование на десктопе).
    - Текстовый блок с направлениями работы и цитатой.
4. **Expertise Summary (Итоговый блок)**:
    - Цветное изображение с мягким затуханием по краям.
    - Объединение компетенций через иконки.
5. **Footer & Navigation**: 
    - Компактный подвал с контактами (Telegram, Почта).
    - Кнопка "Наверх" с плавной анимацией появления.

## ⚙️ Функционал и Интерактивность

- **Адаптивность (Responsive)**:
    - Полная поддержка мобильных устройств.
    - Центрирование заголовков и уменьшение отступов на малых экранах.
    - Оптимизированное отображение блоков (вертикальная колонка vs сетка).
- **Анимации**:
    - `whileInView`: Плавное появление элементов при скролле.
    - `whileHover/whileTap`: Интерактивный отклик кнопок.
    - Плавный скролл (Smooth Scrolling).
- **Интеграции**:
    - Прямые ссылки в Telegram.
    - Ссылка `mailto` для связи по почте.

## 🛠 Установка и запуск

1. Склонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Запустите сервер разработки: `npm run dev`.
4. Для сборки проекта: `npm run build`.
